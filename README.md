## Setup Instructions

### Backend

1. Clone the repository:
```bash
git clone https://github.com/aadyanair/synapse.git
```

2. Navigate to the backend folder:
```
cd article-qna/article-qna
```
3. Configure the Claude API key in your environment:
```
export CLAUDE_API_KEY=your_api_key_here
# or on Windows PowerShell:
setx CLAUDE_API_KEY "your_api_key_here"

```
4. Build and run the project:
```
mvn clean spring-boot:run
```
5. The API will be available at:
```
http://localhost:8080/api/qna
```


### Frontend

1. Navigate to the frontend folder:
```
cd path/to/frontend
```

2. Install dependencies and start the dev server:
```
npm install
npm start
```

3. Access the frontend at:
```
http://localhost:3000
```


## Usage

- Enter the article URL or text in the frontend.

- Type your question.

- Receive the answer generated by Claude AI.

## Notes

- API keys are not included in the repository. Please set your own Claude API key in environment variables.

- The project currently uses Claude AI for QnA. No OpenAI keys are required.

- The text embedding service is pseudo-implemented and can be extended for semantic search or other ML tasks.
